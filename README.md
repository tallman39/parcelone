# ParcelOne — UX Case Study Project

Commercial autonomous drone delivery platform (fictional startup).  
**Launch market:** Dallas–Fort Worth, TX (Phase 1: Frisco / Collin County)

## Project files

| File | Purpose |
|------|---------|

## Lovable build order

1. Command Center (3-panel: Fleet · Map · Mission Detail)
2. Pack & Scan overlay
3. Failed delivery / fault state
4. Orders table
5. Analytics
6. Settings + weather hold
7. Proof of delivery + billing
8. Login + location switcher
